# Tom's Data Onion Solver

**Python script to solve Tom's Data Onion (https://www.tomdalling.com/toms-data-onion/)**

## Installation

```
git clone https://github.com/ola-ct/data-onion.git
pip3 install pipenv
cd data-onion
pipenv install
```

## Run solver

```
pipenv run ./solve.py
```
